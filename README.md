App Summary

Ken's Full Franks is a mobile app that lets you the user build the hot dog your taste buds desire.
The user is allowed to customize their hot dog to their liking as well as add a side and drink
A few of the customizable options are...
- Buns
- Meats
- Toppings( Regular or Premium)
- Sauces

We have...
- New York-Style inpired Hot Dog
- Loaded Chicago inspired
- The Seattle favorite
